Language / Libraries

  • Python3 (backend)
  • Flask - One awesome Python web microframework. The server side of the client/server relationship.
  • Peewee - One epic Python database framework. Manages the storage and retrieval of all the cool stuff.
  • HTML/CSS/Javascript (frontend)
  • Foundation - The most advanced responsive front-end framework in the world. Too awesome for your eyes.
  • jQuery - The most used Javascript framework out there. Used here too. Funny that.
  • jQuery Tubular - Something that puts YouTube videos as a webpage background… hello!
  • Font Awesome - Used to make icons awesome.
  • Selectize - Makes dropdowns super special and not boring.
  • Tipped - Makes tooltips become super special and allows for fancy shmancy HTML.

Any other frameworks seen in the bower components directory are dependencies of one or more of the above frameworks.
